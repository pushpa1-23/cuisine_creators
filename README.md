# 🍽️ Cuisine Creators

Cuisine Creators is a modern web application designed for food lovers to explore, create, and share delicious recipes from different cuisines around the world. The platform provides an interactive and user-friendly experience where users can discover recipes, save their favorites, and contribute their own unique dishes.

---

## 🚀 Features

* 🔍 Search recipes by cuisine, ingredients, or category
* 🍜 Explore recipes from multiple cuisines
* ❤️ Save favorite recipes
* 👨‍🍳 Add and manage custom recipes
* 📱 Responsive and user-friendly UI
* 🔐 User authentication and authorization
* ⚡ Fast and optimized performance

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/pushpa1-23/cuisine_creators.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd cuisine_creators
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Run the Application

```bash
npm start
```

The application will run on:

```bash
http://localhost:3000
```
